# Refund Policy — Gmail Inbox Intelligence

**Last updated**: 2026-05-15

## Free Tier

- 100 threads / month free
- No payment required, no refund applicable
- Trial periods may extend Free quota (announced separately)

## Pro Subscription ($19/month)

### 7-day Refund Window
- First-time subscribers: full refund within 7 days of first subscription if you've used **less than 500 threads metadata**
- Request via Apify support or [Fox 上架時填入 email]
- Refund processed within 5 business days

### After 7 days
- No refunds for monthly subscriptions
- You can cancel anytime; cancellation takes effect at end of current billing cycle
- No partial refunds for mid-cycle cancellation

### Renewal
- Auto-renews monthly unless cancelled
- Apify will email reminder before renewal

## Pay-Per-Result Add-on

- Charges accrued during Actor run are final
- No refund for completed runs
- If a run fails with error 5xx or quota exhausted **before completing**, Apify automatically does not charge

### Failed Runs
- Authentication failures (your OAuth expired) → not charged
- Gmail API quota errors → not charged for unprocessed threads
- Actor code bug (our fault) → fully refunded after support ticket review

## Disputed Charges

- Open a support ticket with: Actor run ID, expected vs actual result, timestamp
- Response SLA: 48 hours
- Escalation: Apify platform support if not resolved within 7 days

## Cancellation

- Pro: cancel via Apify subscription dashboard
- PPR: just stop running the Actor; no recurring charge
